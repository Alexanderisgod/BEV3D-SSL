1. 监督和非监督定义

| 有监督 | **用有标签的数据训练**                                                      |
| --- | ------------------------------------------------------------------ |
| 半监督 | 用无标签的数据训练                                                          |
| 无监督 | 同时用有标签和无标签的数据进行训练                                                  |
| 弱监督 | 用包含噪声的有标签的数据训练                                                     |
| 自监督 | 在无标注的数据上训练， 通过一些方法让模型学习到数据的 inner representation， 再接下游任务 Fine-tune |
